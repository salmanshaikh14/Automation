Automatically refreshes a selected tab at a custom interval.

## Features
- Select any open tab
- Auto refresh page
- Custom interval (seconds)
- Start/Stop controls
- Runs in background

## Install
1. Open edge://extensions or chrome://extensions
2. Enable Developer mode
3. Click Load unpacked
4. Select this folder

## Usage
1. Open tab to refresh
2. Open extension, select tab
3. Set interval
4. Click Start

## Notes
- Very short intervals may log you out
- Frequent refresh may increase CPU
